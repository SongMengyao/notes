## 贮藏 详解

- sourcetree 版

  修改已存在的文件，可以在souretree直接贮藏

  新建立的文件，需要先暂存，在贮藏

  所以，为了避免出现不必要的问题，`建议全部文件暂存之后再贮藏`

  每个贮藏里面包含的文件，都不能单独使用。只能直接使用那个贮藏(那个贮藏里面的所有文件)

  - 贮藏步骤
  1. 暂存所有不提交的文件(旧文件&&新文件)
  2. 点击‘贮藏’ => 填写该贮藏的名字
  3. 完成即可
  - 取贮藏的步骤
  1. 菜单栏选中要取的贮藏，击右键，选择‘使用’
  2. 有冲突的解决冲突，无冲突的刚刚好

  `所有贮藏，可用于任何分支。`比如在a分支建立的贮藏stashing-a，也可以在b分支取出来使用
