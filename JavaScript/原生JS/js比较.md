**` js比较 `**

1. == 和 ===
  - == 是先试图把两边的值转换成相同类型的，然后在比较
  - === 是先比较两边的值的类型，如果类型相同，然后继续比较。否则，返回false
  ```
  eg: 5 == '5'  // true
      5 === '5'  // false
  ```
  `但是，两个Obiect类型的对象是无法进行比较的，值也是无法相互转换的，所以 == 和 === 效果一样，都是false`
  ```
  eg: var a = {
        x: 1
      }

      var b = {
        x: 1
      }

      a == b  // false
      a === b  // false
  ```
  `!= 和 !== 用法类似，只是这个是不等`
  ```
  eg: 5 != '5'  // false
      5 !== '5'  // true
  ```
2. js 逻辑，也就是 `>、<、>=、<=`
