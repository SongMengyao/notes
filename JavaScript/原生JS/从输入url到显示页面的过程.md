**` 从输入url到显示页面的过程 `**
1. DNS 解析，建立 TCP 连接，发送 http 请求 

    (DNS 解析：域名解析，eg：www.baidu.com域名解析到具体的IP地址，比如172.168.24.255)

    (TCP 连接：三次握手)

2. server 接收到 http 请求，处理，并返回

3. 客户端接收到返回的数据，处理数据

    (比如：渲染页面，执行js等)


第一步和第三步，是客户端的事情，第二步是server端的事情。

- https：默认端口443
- http：默认端口80
