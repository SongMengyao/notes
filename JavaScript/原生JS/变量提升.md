**` 变量提升 `**

  js中的变量可以先使用，在声明。`函数及变量的声明都将被提升到函数的最顶部。` eg：

    ```
      x = 10 // 使用
      console.log('x: ', x)  // x: 10
      var x // 声明
    ```

  `js只有声明的变量会提升，初始化的不会。` eg：

    ```
      列1、
      x = 10
      y = 5
      console.log('x, y, x + y: ', x, y, x + y)  // 10, 5, 15
      var x // 声明
      var y = 15 // 初始化y
    ```
    ```
      列2、
      a = 1
      document.write('a 是: ', a) // 1
      document.write('b 是: ', b) // undefined
      var a  // 声明a
      var b = 2 // 初始化b
    ```
